# Tela de Login

![Demonstração](https://i.imgur.com/vp7POiv.png)

## Descrição

Este projeto consiste em uma tela de login com um design moderno e atraente. A interface de usuário foi criada utilizando HTML para a estrutura e CSS para o estilo visual.

## Funcionalidades

- Campo de texto para Email
- Campo de texto para Senha
- Link "Esqueci minha senha"
- Botão "Entrar"
- Link para cadastro de novo usuário

## Tecnologias Utilizadas

- <img src="https://upload.wikimedia.org/wikipedia/commons/6/61/HTML5_logo_and_wordmark.svg" alt="HTML5" height="20"/> HTML
- <img src="https://upload.wikimedia.org/wikipedia/commons/d/d5/CSS3_logo_and_wordmark.svg" alt="CSS3" height="20"/> CSS

## Como Usar

1. Clone o repositório para sua máquina local.
2. Abra o arquivo `index.html` em um navegador da web para visualizar a tela de login.
3. Edite os arquivos conforme necessário para adaptar o projeto às suas necessidades.
